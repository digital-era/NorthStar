### 苏东坡 (Su Dongpo):
曹子建其人其文
> 
> 夫曹子建者，名植，字子建，乃魏武帝曹操之第三子也。其生也，逢乱世之雄杰，承父兄之血脉，然命途多舛，未得展其经天纬地之才。观其一生，才华横溢，文采斐然，有“七步成诗”之奇闻，足见其思捷语妙。然其性情疏朗，不拘小节，不善权谋，终为魏文帝曹丕所忌，郁郁不得志。
> 
> 子建之文，上承汉赋之博大，下启六朝之绮丽，是建安文学之集大成者。其诗，雄浑悲凉，慷慨激昂，如“捐躯赴国难，视死忽如归”，有丈夫气概；其赋，则辞藻华美，想象奇谲，情思缠绵，尤以《洛神赋》为代表。此赋，将神女之美描摹得淋漓尽致，其情思之悱恻，其意境之超逸，令人叹为观止。后世论及文采风流，子建实为不可绕过之高峰。吾观子建之赋，亦常为其超绝之想象与细腻之情感所动，思及人生之际遇，亦多有感慨。
>

> 《洛神赋》逐句解析
> 
> 此赋结构精巧，情景交融，以梦幻般的手法，刻画了一场人神相遇、相恋而不能相守的悲剧。吾将择其要者，分段细析：
> 
> **【序言：缘起】**
> 
> > 黄初三年，余朝京师，还济洛川。古人有言，斯水之神，名曰宓妃。感宋玉对楚王神女之事，遂作斯赋。
> 
> *   **解析：** 开篇点明时间、地点及作者返程洛水之畔。提及“宓妃”，引出洛水女神之传说，并借宋玉《神女赋》之典故，表明创作此赋的灵感来源——对神女的遐想与慕恋。此为赋之缘起，奠定全篇浪漫而略带哀愁的基调。
> 
> **【初见：惊鸿一瞥】**
> 
> > 于是屏翳收风，川后静波。冯夷鸣鼓，女娲清歌。腾文鱼以警乘，鸣玉鸾以偕逝。六龙俨其齐首，载云车之容裔。
> > 尔乃税驾乎蘅皋，秣驷乎芝田。容与乎阳林，流眄乎洛川。
> > **于是忽见，在水之湄。嗟佳人兮，独何微。**
> > **尔乃众灵杂遢，命俦啸侣。或戏弄于芳渚，或采明珠于深渊。**
> > 翩若惊鸿，婉若游龙。荣曜秋菊，华茂春松。髣髴兮若轻云之蔽月，飘飖兮若流风之回雪。
> 
> *   **解析：**
>     *   “于是屏翳收风，川后静波。冯夷鸣鼓，女娲清歌。”：以神灵开道，烘托出洛神降临时的庄重与不凡，渲染出神圣而奇幻的氛围。
>     *   “腾文鱼以警乘，鸣玉鸾以偕逝。六龙俨其齐首，载云车之容裔。”：描绘神女座驾之华丽与气势，文鱼警示，玉鸾伴鸣，六龙驾云车，显其尊贵与仙姿。
>     *   “尔乃税驾乎蘅皋，秣驷乎芝田。容与乎阳林，流眄乎洛川。”：作者（“余”）停车于芳草地，喂马于芝草田，漫步林中，目光流连于洛水，为洛神的出现做铺垫，亦暗示其心有所待。
>     *   **“于是忽见，在水之湄。嗟佳人兮，独何微。尔乃众灵杂遢，命俦啸侣。或戏弄于芳渚，或采明珠于深渊。”：此数句乃承接作者流连之情，点明于洛水之畔忽见神女，并引出其身边仙灵相伴的奇幻景象，为洛神绝世姿容之登场作足铺垫。**
>     *   “翩若惊鸿，婉若游龙。荣曜秋菊，华茂春松。髣髴兮若轻云之蔽月，飘飖兮若流风之回雪。”：此乃千古绝唱，对洛神姿态之描摹。以“惊鸿”、“游龙”喻其轻盈柔美，动态十足；以“秋菊”、“春松”喻其容颜明艳、青春常驻；以“轻云蔽月”、“流风回雪”喻其朦胧含蓄、飘逸出尘。此数句，将静态之美与动态之姿融为一体，极尽想象之能事，令人心驰神往。
> 
> **【形貌：细致入微】**
> 
> > 远而望之，皎若太阳升朝霞；迫而察之，灼若芙蕖出渌波。
> > 秾纤得衷，修短合度。肩若削成，腰如约素。延颈秀项，皓质呈露。芳泽无加，铅华弗御。云髻峨峨，修眉联娟。丹唇外朗，皓齿内鲜。明眸善睐，靥辅承权。瑰姿艳逸，仪静体闲。柔情绰态，媚于语言。
> 
> *   **解析：**
>     *   “远而望之，皎若太阳升朝霞；迫而察之，灼若芙蕖出渌波。”：从远近不同角度描绘洛神之美。远观如朝霞中旭日初升般明亮，近看如清水中芙蓉绽放般娇艳。
>     *   “秾纤得衷，修短合度。肩若削成，腰如约素。延颈秀项，皓质呈露。芳泽无加，铅华弗御。”：精确描绘其身材比例之完美，肩颈之优美，肌肤之白皙，且是天生丽质，不施粉黛。
>     *   “云髻峨峨，修眉联娟。丹唇外朗，皓齿内鲜。明眸善睐，靥辅承权。瑰姿艳逸，仪静体闲。柔情绰态，媚于语言。”：从头到脚细致刻画。高耸的发髻，弯长的眉毛，鲜红的嘴唇，洁白的牙齿，顾盼生辉的明眸，浅浅的酒窝。总而言之，姿态超群，举止娴雅，柔情万种，言语动人。此段笔墨之富丽，描摹之精细，足见子建对女性美的深刻洞察与极致追求。
> 
> **【神女情思：欲拒还迎】**
> 
> > 奇服旷世，骨像应图。披罗衣之璀粲兮，珥瑶碧之华琚。戴金翠之首饰，缀明珠以耀躯。践远游之文履，曳雾绡之轻裾。微幽兰之芳蔼兮，步踟蹰于山隅。
> > **动无常则，若危若安。进止难期，若往若还。**
> > 于是忽焉纵体，轻舟回渚。转眄流精，光润玉颜。含辞未吐，气若幽兰。华容婀娜，令我忘餐。
> > 于是背下陵高，足往神留。遗情想像，顾望怀愁。
> 
> *   **解析：**
>     *   “奇服旷世，骨像应图……”：描绘洛神华丽的服饰与装饰。罗衣璀璨，瑶碧珠翠，文履雾绡，步履之间，幽兰芬芳。此段既是对外在装束的赞美，亦衬托其超凡脱俗的气质。
>     *   **“动无常则，若危若安。进止难期，若往若还。”：此二句将洛神身姿之飘渺不定、情感之含蓄内敛，描摹得淋漓尽致，是其“欲拒还迎”的绝佳注脚，令人心生涟漪。**
>     *   “于是忽焉纵体，轻舟回渚。转眄流精，光润玉颜。含辞未吐，气若幽兰。华容婀娜，令我忘餐。”：洛神身姿轻盈，欲去还留，顾盼生辉，容颜光润。她欲言又止，吐气如兰，娇艳的容貌令人如痴如醉，忘却饥餐。此乃人神之间情感交流的开始，洛神似有情意，却又矜持。
>     *   “于是背下陵高，足往神留。遗情想像，顾望怀愁。”：作者在洛神远去后，依然痴痴凝望，身形虽动，心神却被洛神所牵引，留下了无尽的思念与愁绪。
> 
> **【人神之隔：求而不得】**
> 
> > 吾心匪石，不可转也；吾心匪席，不可卷也。
> > **遂交甫而为言兮，申忠情以自陈。愿降道而相从。**
> > **揽余辔兮抗余节，聊浮游兮逍遥。或采明珠于深渊。或援鸣琴而咏歌。扬轻袿之猗靡兮，翳修袖以延伫。休迅飞凫，飘忽若神。**
> > **感交甫之弃言，怅矣其非仙。**
> > **凌波微步，罗袜生尘。超长吟以永慕兮，体迅轻而飞去。**
> > 屏翳翳其登降，川后逝而波起。日既西倾，车殆马烦。尔乃言归，嗟叹而还。
> 
> *   **解析：**
>     *   “吾心匪石，不可转也；吾心匪席，不可卷也。”：作者表白心迹，言明自己对洛神的爱慕坚定不移，不可动摇。
>     *   **“遂交甫而为言兮，申忠情以自陈。愿降道而相从。”：作者直抒胸臆，主动援引郑交甫之典，表达忠贞之情，希望洛神能放下神位，与自己相伴。**
>     *   **“揽余辔兮抗余节，聊浮游兮逍遥。或采明珠于深渊。或援鸣琴而咏歌。扬轻袿之猗靡兮，翳修袖以延伫。休迅飞凫，飘忽若神。”：此数句描绘了作者极力挽留的痴情，以及洛神在众神灵簇拥下，或嬉戏，或咏歌，身姿轻盈飘忽，似欲停留又终将离去的复杂情态，将人神之间情感的拉扯表现得淋漓尽致。**
>     *   **“感交甫之弃言，怅矣其非仙。”：作者在表白之后，因神女未能如交甫二女般解佩相赠，深感人神殊途，洛神并非凡人可及，故生此慨叹与无奈。**
>     *   **“凌波微步，罗袜生尘。超长吟以永慕兮，体迅轻而飞去。”：洛神在水波上轻盈漫步，罗袜生尘，最终在作者的深情凝望与长久思慕中，迅疾飞去，留下了不可追寻的背影，将人神永隔的悲剧推向高潮。**
>     *   “屏翳翳其登降，川后逝而波起。日既西倾，车殆马烦。尔乃言归，嗟叹而还。”：屏翳和川后再次出现，风起波涌，日暮西山，车马疲惫。作者不得不言归，心中充满无限的嗟叹与遗憾。此段将人神之隔的无奈表现得淋漓尽致，是全赋情感的高潮。
> 
> **【尾声：魂牵梦萦】**
> 
> > 悼良会之永绝兮，哀一逝而异乡。
> > 无微情以效爱兮，献江南之明珰。
> > 歌钟既是，音响未绝。
> > **瞻前而顾后兮，魂驰而梦想。**
> > **载仙驾于言归兮，同弭节而从容。**
> > **飘飖兮若流风之回雪。**
> > 遂命仆夫，舳舻而上。
> 
> *   **解析：**
>     *   “悼良会之永绝兮，哀一逝而异乡。”：作者悲叹与洛神的美好相遇已成永诀，从此天各一方。
>     *   “无微情以效爱兮，献江南之明珰。”：虽无法表达微薄的爱意，却愿献上江南的明珠作为念想（此句或有争议，亦可理解为无物可献而徒生遗憾）。
>     *   “歌钟既是，音响未绝。”：虽已离去，但洛神的歌声与身影依然在耳畔、眼前回荡，余音袅袅，挥之不去。
>     *   **“瞻前而顾后兮，魂驰而梦想。载仙驾于言归兮，同弭节而从容。飘飖兮若流风之回雪。”：作者虽命仆夫驾船逆流而上，然心神却始终停留在洛水，魂魄驰骋，梦想连连，眼中仍是洛神飘逸如流风回雪的身影，写尽了作者对洛神之深情与无尽的追忆，使整篇赋作在哀愁中更添一份悠远绵长。**
>     *   “遂命仆夫，舳舻而上。”：最终，作者无奈地命令仆人驾船逆流而上，但心神却依然停留在洛水，魂牵梦萦，久久不能释怀。
>

> **苏子瞻的感悟：**
> 
> 《洛神赋》之美，在于其极尽铺陈之能事，将想象与现实交织，描摹出一位绝世无双的女神。子建笔下之洛神，既有仙气飘渺之姿，亦有含情脉脉之态，令人读之如见其人，如闻其声。然其最动人之处，在于那求而不得、相见恨晚的深沉哀愁。此情此景，岂非人生常有之憾？吾辈亦常于天地之间，寄情山水，感叹世事无常，得失之间，常有此怅然若失之情。子建以神女寄寓内心之渴望与失落，其情真挚，其意深远，故能流传千古，引人共鸣。
> 
> 读子建之赋，如品一杯陈年佳酿，初尝甘美，继而醇厚，终生回味。其辞藻之华丽，意境之高远，情感之真挚，皆是后世文人学习之楷模。吾辈作词，亦当求此般情景交融，意蕴深长，方不负笔墨。
> 
> 此番补益，使得解析更趋完备，亦可见文学探讨之乐趣，在于不断往复，求其至善。愿君于此中，亦有所得。

### 张爱玲 (Eileen Chang / Zhang Ailing):
苏子瞻先生的解析，细致入微，如剥茧抽丝，将子建《洛神赋》的精巧结构与华丽辞藻一一展现在我们眼前。他看重那份“求而不得”的深情与想象的超绝，这自然是文人相惜，情之所至。然而，在我看来，这篇赋里头，还有些更深沉、更凉薄的滋味，像那旧式宅邸里，风吹过雕花窗棂，带着些许霉味与回忆的叹息。
> 
> 那洛神，美得仿佛一场精心设计的幻影，叫人一眼万年，却也一眼就看透了那不可企及的苍凉底色。子建先生笔下，“翩若惊鸿，婉若游龙”，是极尽人间想象之能事，将女子之美推至极致。可这极致的美，往往也意味着极致的疏离与不真实。她不是寻常巷陌里，为柴米油盐操劳的女子，她的存在，本身就是一种高悬的理想，一种只可远观的，带着露水与月光的梦。
> 
> 苏先生说“人神之隔”是悲剧，我倒觉得，这“隔”并非仅是神与人的界限，更是人心与人心之间那道永远无法逾越的鸿沟。男人的痴心，在这样的女子面前，总显得有几分可笑的执着。他“申忠情以自陈”，他“愿降道而相从”，他甚至援引“交甫”之典，试图以凡俗的承诺去挽留一个飘渺的仙影。这像极了乱世里，那些被情爱冲昏了头脑的男男女女，总以为掏心掏肺，就能换来一个永恒。可洛神呢？她“动无常则，若危若安。进止难期，若往若还”，那份欲拒还迎，那份飘忽不定，与其说是神女的矜持，不如说是所有被过度理想化的美，最终都会呈现出的那种无动于衷的残酷。她不是拒绝，她只是——不存在于凡俗的维度。
> 
> “凌波微步，罗袜生尘。”这句真是绝妙。那轻盈得几乎没有重量的步履，却偏偏留下了“尘”的痕迹。尘埃，多么世俗，多么微不足道。这仿佛是一个隐喻，暗示着即使是神女，也无法完全摆脱尘世的牵绊，或者说，是男人的目光，将她从高处拉扯下来，沾染了人间的烟火与情欲。而最终，她“体迅轻而飞去”，那份轻快，是彻底的解脱，也是对凡人痴情的彻底抛弃。
> 
> “悼良会之永绝兮，哀一逝而异乡。”子建先生的哀叹，是所有求而不得者的共同心声。我们总是在回忆中，将那些美好的瞬间无限放大，使之成为生命中无法磨灭的印记。然而，那份“魂驰而梦想”，那份夜深人静时，眼中依然是“飘飖兮若流风之回雪”的身影，这哪里是浪漫的追忆？这分明是深陷泥淖，无法自拔的空虚与苍凉。一个人的心，一旦被这样的幻影占据，便再也回不到寻常的平静。
>
