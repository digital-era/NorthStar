## 东山精密2026-2028年经营分析与预测深度研究

### 摘要

本报告对东山精密2026-2028年经营情况进行全面分析，重点关注公司通过收购索尔思光电和GMD集团实现的业务转型，以及在AI算力基础设施领域的战略布局和业绩增长潜力。基于公司2025年年报及2026年Q1业绩表现，结合AI算力基础设施市场的高速增长趋势，我们预测公司未来三年将实现营收和利润的爆发式增长。2026年，公司营收有望达到650亿元左右，同比增长约62%；归母净利润约75亿元，同比增长437%。2027年，公司营收预计达到850亿元，同比增长31%；归母净利润约130亿元，同比增长73%。2028年，随着3.2T光模块量产和CPO技术补充需求，公司营收有望突破1100亿元，归母净利润约200亿元，同比增长54%。然而，公司也面临硅光技术替代风险、原材料短缺风险和高负债率带来的财务压力等挑战。我们将从业务结构转型、财务状况变化、AI算力基础设施市场需求与竞争格局以及未来三年业绩预测与风险评估四个维度进行全面分析，为投资者提供决策参考。

**关键词**：东山精密、索尔思光电、光模块、AI算力基础设施、PCB、垂直整合、业绩预测、风险评估

### 一、公司概况与业务转型

#### 1.1 公司基本情况

东山精密(002384.SZ)成立于1980年，2010年在深交所上市，是一家专注于精密制造的高科技企业。公司业务涵盖电子电路、光电显示和精密制造三大板块，产品广泛应用于消费电子、新能源汽车、通信设备及AI等行业。以收入规模计算，公司连续多年柔性线路板(FPC)排名全球第二，PCB排名全球第三，是全球领先的电子电路解决方案提供商。

2025年，公司通过一系列战略并购完成了向AI算力基础设施领域的转型：
- 2025年10月，公司完成对索尔思光电100%股权的收购，正式切入光通信市场
- 同年，公司完成对法国GMD集团的收购，进一步拓展了海外业务布局

#### 1.2 业务结构转型

**从传统制造向AI算力基础设施转型**是东山精密2025-2026年最重要的战略转变。公司通过收购索尔思光电，构建了"光芯片+光模块"垂直整合一体化的IDM经营模式，同时结合旗下Multek的高端PCB制造能力，形成了全球稀缺的"PCB+光芯片+光模块"全栈布局能力。

具体业务转型路径如下：

1. **光模块业务并表**：索尔思光电于2025年10月正式纳入东山精密合并报表范围，其产品覆盖从10G到800G、1.6T及以上速率的各类光模块，采用IDM模式实现核心环节自主可控，是国内少数具备100G和200G光芯片量产能力的企业。

2. **高端PCB产能扩张**：公司投入超10亿美元扩充高端PCB产能，Multek具备78层及以上超高多层与7阶厚板HDI PCB制造能力，是英伟达GB200服务器78层PCB的独家供应商。

3. **客户结构优化**：公司从消费电子和传统PCB向AI服务器和数据中心领域转型，成功进入英伟达、Meta、AWS等全球顶级AI算力客户的供应链。

#### 1.3 2025年业绩表现

2025年，公司实现营业收入401.25亿元，同比增长9.12%，首次突破400亿元大关；归母净利润13.86亿元，同比增长27.67%。业务结构显著优化：

### 二、2026年Q1业绩分析

#### 2.1 业绩概览

2026年第一季度，东山精密实现营业收入131.38亿元，同比增长52.72%，环比增长0.64%；归母净利润11.10亿元，同比增长143.47%，环比增长581.80%；扣非归母净利润10.59亿元，同比增长166.99%，环比扭亏，扣非/归母比95.44%，利润质量优异。**单季净利润已超过2025年全年净利润的80%**，业绩爆发力惊人。

毛利率19.33%，同比提升5.20个百分点，环比提升4.62个百分点；净利率8.56%，同比提升3.25个百分点，环比提升7.26个百分点。盈利能力显著提升。

#### 2.2 业绩增长驱动力分析

**2026年Q1业绩高增的核心驱动力来自索尔思光电及GMD集团并表，叠加全球AI算力需求爆发催化光模块行业景气上行**。具体来看：

1. **光模块业务爆发式增长**：索尔思光电光模块业务收入同比翻倍，800G光模块已向Meta出货，Oracle、Microsoft、xAI等客户导入进展顺利，AWS及NVDA 1.6T产品送样推进中。据行业调研数据，2026年Q1索尔思光电800G光模块出货约30万只，400G光模块同样出货30万只，其中近90%产品出口海外，牢牢绑定北美顶级云厂商。

2. **AI相关利润贡献占比过半**：Q1 AI相关业务利润贡献占比已超50%，成为公司核心盈利支柱。

3. **高端PCB业务稳步增长**：Multek具备78层以上超高多层与7阶厚板HDI PCB制造能力，已累计投入超13亿美元扩充高端产能，服务器领域GPU/ASIC客户打样进展顺利，收入与利润同比稳步增长。

4. **传统业务保持稳定**：电子电路产品(含FPC)、精密组件(含GMD)及光电显示模组出货保持稳定，为新兴AI硬件业务的扩张提供了基本面支撑。

#### 2.3 财务状况分析

尽管业绩增长迅猛，但公司财务状况也面临挑战：

1. **负债率大幅上升**：截至2025年末，公司资产负债率达63.98%，较2024年增加5.02个百分点。2026年Q1，资产负债率进一步升至69.41%，处于行业较高水平。**高负债率主要源于收购索尔思光电和GMD集团新增借款**。

2. **财务费用暴增**：2026年Q1财务费用达2.48亿元，同比激增3479.28%，主要系上年并购索尔思光电等资产新增并购贷款，导致利息费用大幅增加，同时汇兑损失超1亿元。

3. **现金流与利润背离**：2026年Q1经营现金流净额为11.27亿元，同比下滑17.48%。应收账款增加15亿元、存货增加8亿元，现金流增长依赖压供应商账款与预收款，回款质量有待提升。

4. **在建工程大幅增加**：截至2026年Q1末，公司在建工程达34.48亿元，较2025年末增长46.96%，主要系公司围绕下游AI数据中心需求的爆发，加大了光模块(含光芯片)及AI PCB的产能扩张，以应对客户订单和交付要求。

### 三、AI算力基础设施市场需求与竞争格局

#### 3.1 AI算力基础设施市场需求分析

**全球AI算力基础设施市场正处于爆发式增长阶段**，为东山精密的转型提供了巨大机遇：

1. **AI服务器出货量高速增长**：2025年全球AI服务器机架出货量约为2.9万台，而2026年这一数字将直接跃升至7-8万台，实现翻倍增长。美银证券预测，2026年全球服务器市场规模将达到7569亿美元，同比增长75%，其中AI服务器将扮演绝对的增长引擎，其市场规模预计将达到4957亿美元，同比激增104%。

2. **光模块需求激增**：据高盛亚太区科技分析师Allen Chang预测，2026年全球AI光模块市场规模预计将达到260亿美元，年增速高达57%。到2026年，800G和1.6T高速光模块的合计市场规模将占整体市场的约64%。

3. **PCB需求结构变化**：AI服务器对PCB的要求发生根本性变化，从传统服务器8-12层板升级至20层以上电路板，高端型号甚至达到40至80层。AI服务器单台PCB价值量从传统服务器的3000元跃升至1.5万-4万元，增长了5-10倍。

4. **技术迭代加速**：光模块的升级周期已从过去的4年大幅压缩至2年左右。800G当前是主流，而1.6T光模块在2026年已进入商业化放量元年，2027年需求预计从1000万只爆发至3000万只。

#### 3.2 光模块市场竞争格局

光模块市场竞争格局呈现以下特点：

1. **头部集中度提升**：中际旭创2026年Q1全球光模块市占率达23%，连续三年稳居全球第一。在800G高速光模块领域，市占率超过40%；在1.6T光模块领域，市占率更是超过50%-60%。

2. **技术路线分化**：
   - **硅光路线**：以中际旭创为代表，硅光模块良率达95%，功耗降低40%，成本比传统光模块方案低30%，市占率快速提升。
   - **EML芯片路线**：索尔思光电采用IDM模式，具备光芯片自研自产能力，为国内唯一可自供EML芯片并大规模量产的企业，2025年上半年全球光芯片产量排名第七，市占率为4.4%。

3. **供需失衡加剧**：Lumentum的EML激光器芯片等核心元器件持续短缺，订单已排至2028年，供需缺口从20%扩大至25-30%。高盛预测，2027年二季度后，1.6T/3.2T高速光模块、光引擎、高端光芯片将出现全球性严重短缺。

4. **客户集中度高**：中际旭创对英伟达的依赖度达60%，索尔思光电的客户结构也呈现高度集中特征，主要服务于北美顶级云厂商。

#### 3.3 AI PCB市场竞争格局

AI PCB市场竞争格局呈现以下特点：

1. **技术门槛高**：AI服务器需要70层以上高多层PCB和8+N+8及以上HDI PCB，技术壁垒极高。目前全球仅少数厂商具备此类生产能力，如沪电股份、胜宏科技和东山精密Multek等。

2. **市场格局**：沪电股份在22层及以上PCB全球市占率达25.3%，在数据中心领域PCB收入全球市占率达10.3%。胜宏科技AI服务器PCB全球市占率达13.8%，2025年高多层与高阶HDI收入占比已升至68%。东山精密Multek是英伟达GB200服务器78层PCB的独家供应商，但整体市场份额仍低于行业龙头。

3. **技术迭代加速**：AI服务器设计正迎结构性转变，从英伟达Rubin平台无缆化架构，到云端大厂自研ASIC服务器的高层HDI设计，PCB不再只是电路载体，而是成为算力释放核心层，PCB正式进入高频、高功耗、高密度时代。

4. **产能扩张竞争**：据不完全统计，国内厂商高端PCB扩产规划已近700亿元。胜宏科技计划投资200亿元，沪电股份追加55亿元，鹏鼎控股在淮安投资110亿元，行业竞争日趋激烈。

2. **光模块业务高增长**：索尔思光电光模块业务收入2026-2028年年均复合增长率达100%以上，其中800G产品2026年出货量达500万只，1.6T产品2027年出货量达1000万只。

### 五、潜在风险因素分析

#### 5.1 技术迭代风险

**硅光技术替代EML芯片路线**是东山精密面临的最大技术风险：

1. **硅光技术渗透率快速提升**：根据LightCounting预测，2026年硅光技术在光模块领域的渗透率已突破50%，并呈加速上升趋势。

2. **技术代差风险**：中际旭创等硅光技术路线厂商在成本控制和良率方面具有显著优势，可能对采用EML芯片路线的索尔思光电形成竞争压力。

3. **研发投入压力**：为应对技术迭代，索尔思光电需要持续加大研发投入，提升硅光技术能力，可能对短期盈利能力造成影响。

#### 5.2 供应链与原材料风险

**光芯片原材料短缺**是东山精密面临的第二大风险：

1. **磷化铟衬底供需失衡**：光模块业务所需的磷化铟衬底、薄膜铌酸锂等关键材料供需缺口扩大，全球产能仅能满足需求的50%，若采购受阻将直接影响光模块产能释放。

2. **产能扩张周期长**：光芯片(如EML/CW)产能建设周期长(24-36个月)，索尔思光电计划2026年底前实现400G EML量产，产能扩至22KK/月，2027年全年产能达3亿颗。若产能释放不及预期，可能影响订单交付能力。

3. **原材料价格波动**：PCB业务所需核心原材料价格波动较大，直接影响生产成本，若原材料价格持续上涨，可能对毛利率造成压力。

#### 5.3 财务风险

**高负债率带来的财务压力**是东山精密面临的第三大风险：

1. **资产负债率持续攀升**：截至2025年末，公司资产负债率达63.98%，较2024年增加5.02个百分点。2026年Q1进一步升至69.41%，短期借款约80亿元、一年内到期非流动负债近35亿元，短期偿债压力较大。

2. **财务费用大幅增加**：2026年Q1财务费用达2.48亿元，同比激增3479.28%，主要系上年并购索尔思光电等资产新增并购贷款，导致利息费用大幅增加，同时汇兑损失超1亿元。

3. **现金流压力**：2026年Q1经营现金流净额同比下滑17.48%至11.27亿元，应收账款增速(2025年+27.78%)远超营收增速(2025年+9.12%)，存货大幅增加，现金流与利润出现背离。

#### 5.4 客户集中度风险

**对头部AI算力客户的高度依赖**是东山精密面临的第四大风险：

1. **客户结构集中**：索尔思光电主要客户为北美顶级云厂商，如英伟达、Meta、AWS等，客户集中度高，若主要客户需求波动，可能对业绩造成较大影响。

2. **订单能见度与实际交付的差距**：虽然公司表示订单能见度显著提升，长单金额超12亿美元，部分订单已排至2027-2028年，但实际交付仍需考虑产能爬坡和客户验收进度等因素。

3. **技术路线转换风险**：英伟达等客户正在推进CPO(共封装光学)等新技术路线，若技术路线转换速度超预期，可能影响传统光模块需求。

### 六、投资价值与风险评估

#### 6.1 投资价值分析

**东山精密的核心投资价值在于其稀缺的"光芯片+光模块+AI PCB"全栈布局能力**，这一能力使其在AI算力基础设施领域具备独特竞争优势：

1. **垂直整合优势**：索尔思光电采用IDM模式实现"光芯片+光模块"一体化，是国内唯一可自供EML芯片并大规模量产的企业，芯片自给率超99%，成本较外购降低30%以上，这是其高毛利的根本来源。

2. **技术壁垒高**：公司具备生产高达78层高多层PCB和8+N+8及以上HDI的产能，已掌握70层以上高多层正交背板技术，替代传统高速线缆，提升信号完整性、可操作性与可靠性。

3. **客户资源优质**：公司已成功进入英伟达、Meta、AWS等全球顶级AI算力客户的供应链，为未来业绩增长提供了坚实基础。

4. **行业景气度高**：高盛预测，光通信行业未来三年将进入"产能为王"的硬短缺周期，中国光模块与光芯片产业链迎来确定性最强的黄金窗口。

#### 6.2 风险收益平衡分析

基于上述分析，我们对东山精密的投资价值与风险进行平衡评估：

1. **短期投资价值**：2026-2027年，公司业绩增长确定性强，光模块业务放量和AI PCB产能释放将推动营收和利润高速增长，短期投资价值显著。

2. **中期投资风险**：2027-2028年，硅光技术渗透率可能进一步提升，若索尔思光电在硅光技术路线上的布局不及预期，可能面临技术替代风险，中期投资风险增加。

3. **长期投资前景**：若公司能够在3.2T光模块和CPO技术领域保持竞争力，长期投资前景广阔，但需密切关注技术路线转换和行业竞争格局变化。

4. **估值水平**：截至2026年5月，公司市盈率(PE)TTM已高达165倍，明显高于行业平均水平，估值已提前透支未来一两年的高增长预期。

#### 6.3 投资建议

基于对东山精密2026-2028年经营情况的全面分析，我们提出以下投资建议：

1. **短期投资者**：可关注公司业绩兑现情况，尤其是光模块业务和AI PCB业务的产能释放进度和订单执行情况，短期业绩增长确定性强。

2. **中长期投资者**：需密切关注硅光技术路线进展和公司技术布局，若索尔思光电在硅光技术领域取得突破，中长期投资价值将更加凸显；反之，若技术路线转换不及预期，中长期投资风险将显著增加。

3. **风险偏好**：公司当前估值已充分反映未来高增长预期，适合风险偏好较高的投资者，但需警惕业绩不及预期或行业景气度回落带来的估值回调风险。

4. **投资时机**：建议关注公司产能释放进度和订单执行情况，若产能释放顺利且订单执行稳定，可考虑分批建仓；若产能释放延迟或订单执行不及预期，建议谨慎观望。

### 七、结论与展望

**东山精密已成功从一家传统精密制造企业，转型为横跨消费电子、新能源汽车与AI算力基础设施的智能互连方案解决商**。通过收购索尔思光电和GMD集团，公司构建了"光芯片+光模块+AI PCB"的全栈布局能力，这一能力使其在AI算力基础设施领域具备独特竞争优势。

**未来三年，公司业绩增长确定性强**。2026年，公司营收有望达到650亿元左右，同比增长约62%；归母净利润约75亿元，同比增长437%。2027年，公司营收预计达到850亿元，同比增长31%；归母净利润约130亿元，同比增长73%。2028年，随着3.2T光模块量产和CPO技术补充需求，公司营收有望突破1100亿元，归母净利润约200亿元，同比增长54%。

**然而，公司也面临显著风险**。硅光技术渗透率快速提升可能对EML芯片路线构成挑战，磷化铟衬底等关键材料供需缺口扩大可能影响产能释放，高负债率带来的财务压力需要密切关注，对头部AI算力客户的高度依赖可能增加业绩波动风险。

**展望未来**，东山精密的核心竞争力将取决于其在光芯片和光模块领域的技术迭代能力、产能扩张速度以及与AI算力客户的长期合作关系。若公司能够在3.2T光模块和CPO技术领域保持竞争力，长期投资前景广阔；若技术路线转换不及预期，业绩增长可能面临瓶颈。

**总体而言，东山精密的转型之路正以超预期的业绩表现不断验证**，市场对其"光+PCB"双轮驱动的增长模式充满期待。然而，高估值已提前透支未来增长预期，投资者需密切关注业绩兑现情况和技术路线转换风险，做出理性投资决策。
